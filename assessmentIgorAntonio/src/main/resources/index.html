Apresentação do Projeto

Bem-vindo à nossa plataforma de Reservas Online! Aqui está uma visão geral do que oferecemos:

Finalidade:

Esse projeto visa criar um sistema de reserva de hoteis ou qualquer outro tipo de hotelaria!

Objetivos:

  • Criar Hotéis
  • Criar Acomodações (Andares)
  • Criar Quartos para as Acomodações
  • Registrar Clientes
  • Realizar Diferentes reservas de um cliente em algum dos Hoteis a depender da disponibilidade

Funcionalidades Principais:

  • HOTEIS

  • GET localport:8080/hoteis - retorna a lista de hoteis disponiveis.
  • POST localport:8080/hoteis - inclui um hotel na lista de hoteis disponiveis.
  • DELETE localport:8080/hoteis/idDoHotel - exclui um hotel da lista de hoteis disponiveis.
  • GET localport:8080/hoteis/idDoHotel - recupera um hotel pelo id, da lista de hoteis disponiveis.

  • CLIENTES

  • GET localport:8080/clientes - retorna a lista de clientes disponiveis.
  • POST localport:8080/clientes - inclui um cliente na lista de clientes.
  • DELETE localport:8080/clientes/idDoCliente - exclui um cliente da lista de clientes.
  • GET localport:8080/clientes/idDoCliente - recupera um cliente pelo id, da lista de clientes.

  • ACOMODAÇÕES

  • GET localport:8080/acomodacoes/idDoHotel - retorna a lista de acomodações de um hotel.
  • POST localport:8080/acomodacoes/idDoHotel/idDaAcomodacao - inclui uma acomodação na lista de acomodações de um hotel.
  • DELETE localport:8080/acomodacoes/idDoHotel/idDaAcomodacao - exclui uma acomodação da lista de acomodações de um hotel.
  • GET localport:8080/acomodacoes/idDoHotel/idDaAcomodacao - recupera uma acomodação pelo id da lista de acomodações de um hotel.

  • QUARTOS

  • GET localport:8080/quartos/:idDoHotel/:idDaAcomodacao - retorna a lista de quartos de uma acomodação em um hotel.
  • POST localport:8080/quartos/:idDoHotel/:idDaAcomodacao - inclui um quarto na lista de quartos de uma acomodação em um hotel.
  • DELETE localport:8080/quartos/:idDoHotel/:idDaAcomodacao/:idDoQuarto - exclui um quarto da lista de quartos de uma acomodação em um hotel.
  • GET localport:8080/quartos/:idDoHotel/:idDaAcomodacao/:idDoQuarto - recupera um quarto pelo id da lista de quartos de uma acomodação em um hotel.

  • RESERVAS

  • POST localport:8080/idDoCliente/reservar - realiza uma reserva em um hotel a partir do id do cliente fornecido.

Inicialização do Projeto

  1. Run JavaApplication
  2. Acesse: localhost:8080/
  3. Crie um hotel através do POSTMAN
  4. Crie uma acomodação em um hotel através do POSTMAN
  5. Crie um quarto para uma acomodação em um hotel através do POSTMAN
  6. Registre um cliente através do POSTMAN
  7. Realize uma reserva através do ID do cliente em um Hotel pelo POSTMAN
  8. Não se esqueça de verificar se as entidades foram criadas em ordem lógica
